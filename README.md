# 📝 To-Do List Application (Python)

## 📌 Project Description
This is a simple To-Do List application developed using Python.  
Users can add tasks, view tasks, delete tasks, and exit the application.

## ✨ Features
- Add New Task
- View All Tasks
- Delete Task
- Exit Program
- Menu-driven program using `while True`

## 🛠 Technologies Used
- Python

## 📂 Project Structure
```
todo.py
README.md
```

## ▶️ How to Run

1. Install Python
2. Open terminal / command prompt
3. Navigate to project folder
4. Run:

```bash
python todo.py
```

## 📋 Menu Options

```text
1. Add Task
2. View Tasks
3. Delete Task
4. Exit
```
